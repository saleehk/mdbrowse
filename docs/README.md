# mdbrowse-cli Demo

Welcome to the **mdbrowse-cli** demo directory! This showcases what mdbrowse-cli can do.

## Features at a Glance

- **File tree sidebar** — Navigate all your files
- **Beautiful markdown** — GFM tables, code blocks, math, diagrams
- **Syntax highlighting** — VS Code-quality via Shiki
- **Live reload** — See changes instantly
- **Edit mode** — Modify files right in the browser
- **Search** — Find anything with Ctrl+K
- **Dark/light theme** — Toggle with one click

## Quick Start

```bash
npx mdbrowse-cli .
```

> That's it. One command. No config.
